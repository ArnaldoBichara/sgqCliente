Funções de Execução:

  • Atribuir Atividade:
    • Permite verificar a lista de atividades do seu workgroup e atribuir uma Atividade para si
  • Minhas Atividades:
    • Permite verificar a lista de minhas atividades e registra a execução de uma Atividade
  • Registrar Problema:
    • Permite o registro de um Problema relacionado a Qualidade
  • Lista de Problemas:
    • Lista os problemas em aberto relacionados a Qualidade

Funções de Cadastramento:

  • Atividades Cadastradas:
    • Lista o cadastro de atividades de qualidade
  • Cadastrar Atividade:
    • Inclui Atividade no Cadastro
  • Problemas Cadastrados:
    • Lista o cadastro de problemas de qualidade
  • Cadastrar Problema:
    • Inclui Problema no Cadastro

Funções de Execução:

  • AtvGrupo:
    • Lista as atividades do seu grupo de trabalho e permite atribuir uma Atividade para si
  • MinhasAtv:
    • Lista as atividades que você atribuiu para si e permite registrar a execução de uma Atividade
  • RegProbl:
    • Permite o registro de um Problema relacionado a Qualidade

Relatórios:

  • ProblAbertos:
    • Lista os problemas em aberto relacionados a Qualidade
  • AtvExec:
    • Lista as atividades já executadas

Cadastramento:

  • AtvCad:
    • Lista as atividades cadastradas
  • CadAtv:
    • Cadastrar Atividade
  • ProblCad:
    • Lista o cadastro de problemas de qualidade
  • CadProbl:
    • Cadastrar um Problema
